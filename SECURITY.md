# Security Policy

## Supported Versions
At this stage of development (MVP/Prototype), we are focusing on the core infrastructure.

| Version | Supported          |
| ------- | ------------------ |
| MVP     | :white_check_mark: |
| < MVP   | :x:                |

## Reporting a Vulnerability
We take the security of MeshSats seriously. While the project focuses on offline-first payments with honest trust assumptions, securing the settlement and sync layers is paramount.

If you discover a security vulnerability within MeshSats, please do **not** open a public issue. Instead, report it privately to the project maintainers.

Please include:
*   A description of the vulnerability.
*   Steps to reproduce the issue.
*   Potential impact.

We will acknowledge receipt of your report and provide an estimated timeline for a fix or mitigation. 

Thank you for helping keep MeshSats and the Bitcoin community secure.
